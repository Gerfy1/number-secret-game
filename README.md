# Jogo do Número Secreto

este projeto é um pequeno jogo de adivinhação desenvolvido durante os cursos de lógica de programação da Oracle Next Education em parceria com a Alura  
O objetivo é treinar raciocínio lógico utilizando um algoritmo de "maior ou menor" até que o jogador descubra o número correto

## Sobre o projeto

O jogo gera automaticamente um número secreto dentro de um intervalo definido
O usuário tenta adivinhar o valor e recebe dicas informando se o número inserido é maior ou menor que o número secreto.  

Este projeto foi utilizado como base para estudar:

- Estruturas condicionais  
- Laços de repetição  
- Manipulação da árvore DOM  
- Funções e modularização  
- Interatividade básica com JavaScript

## Como executar

1. Baixe ou clone o repositório.  
2. Abra o arquivo `index.html` no navegador.  
3. O jogo inicia automaticamente e as instruções aparecem na tela.

## Objetivo do projeto

Este trabalho foi criado para praticar:

- Estruturas lógicas básicas  
- Manipulação de eventos  
- Separação entre HTML, CSS e JS  
- Primeiro contato com algoritmos simples de comparação
